# react-native-telephony-assistant

USSD and SMS manager for React Native

## Installation

```sh
npm install react-native-telephony-assistant
```

## Usage

```js
import { multiply } from "react-native-telephony-assistant";

// ...

const result = await multiply(3, 7);
```
